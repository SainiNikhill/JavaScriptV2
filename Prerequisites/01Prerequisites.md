# 1. What is Programming language ?
A programming language is a special set of instructions that we use to tell a computer what to do.
Humans speak English, Hindi, etc.
Computers speak binary (0s and 1s).

A programming language acts like a bridge between humans and computers so we can give instructions in an easier way.

---

# 2. What is High level Language ?
A high-level language is easy for humans to read, write and understand.
- looks close to english
- Hides complex hardware details(CPU instructions)
- Easy to learn and use.

---

# 3. what is Dynamic language ?
A Dynamic language decides things at runtime(when the program is runnning), not at compile time.

#### Characterstics:
- No need to specify the data types
- Variables can change type anytime.
- More flexible
- Sometimes slower than static languges.

--- 

# 4. What is Interpreted language ?
An interpreted language run line by line using an interpreter, not compiled to machine code beforehand.
#### Characterstics:
- No separate compilation step
- Runs directly in an environment (like a browser or Python interpreter)
- Easier to test and debug
- Slightly slower than compiled languages

---

# 5. What is JavaScript?

Javascript is a high level, dynamic and interpreted language used to make websites interactive.
It runs directly inside the browser or in a backend environment like NodeJs.

# 7. Why Was JavaScript Created?

In the early internet days, web pages were static.
If you clicked something, the entire page needed to reload.

The goal was to create a language that:
1. Runs inside the browser
2. Makes web pages interactive
3. Is easy for beginners
4. Works together with HTML/CSS

JavaScript allowed developers to:
1. validate forms without reloading
2. show popups
3. update small parts of a webpage instantly
4. interact with users in real time
5. This completely changed the web.

# 8. Words vs Keywords

A word is any name or text you write in a program.

it can be:
- variable name
- function anme
- class name
- or just any text

A word has no special meaning to the programming language unless you define it.


## keyword

A keyword is a special reserved word in a programming language.

It has a fixed meaning and Specific purpose.

You cannot use a keyword as:

- A variable name
- A function name
- or any custom name

JS keywords are:
- let
- var
- const
- if
- else
- for
- while
- return
